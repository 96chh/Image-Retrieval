# Image-Retrieval
- 基于感知哈希算法实现图像检索
- 使用 Python3 & Qt5 实现，其中 UI 部分使用 qt designer 实现
- 依赖有：imagehash, pyqt5
- 命令行输入：`python index.py`即可运行
